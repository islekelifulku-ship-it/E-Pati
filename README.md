# E-Pati 🐾

E-Pati is a pet management application developed with Kotlin in Android Studio.

## Features

- User Login & Registration
- Pet Management
- Health Tracking
- Veterinary Appointments
- Notification System
- Statistics Dashboard
- Calorie & Food Calculator

## Technologies Used

- Kotlin
- Android Studio
- RecyclerView
- SharedPreferences
- Gson
- Material Design Components
- Android Notifications

## Project Purpose

The aim of this project is to help pet owners easily manage and track their pets' daily needs, health records, and appointments through a mobile application.
